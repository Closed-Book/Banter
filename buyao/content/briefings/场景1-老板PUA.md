# ⚔️ 场景 1：老板PUA你压薪

---

## 📍 场景描述

**时间**：周四下午 3 点，年终 review 季
**地点**：公司小会议室，只有你和老板两个人
**你是谁**：入职两年的视觉设计师，月薪 8K，今年绩效评级为「优秀」
**发生了什么**：老板张总约你年终一对一谈话。表面上是关心你的成长，但你知道——他每年都用"公司不容易"压住涨薪。今年你做了三个爆款项目，数据很好看，但张总的态度依然暧昧。

---

## 🎯 你的目标

**拿到至少 10% 的涨薪**（月薪从 8K → 8.8K+），不被 PUA 打回去。

### 通关条件

| 等级 | 条件（数值阈值） | 含义 |
|------|-----------------|------|
| **S** | 目标达成率 ≥ 90 且 主动权 ≥ 70 | 不仅拿到了涨薪，还让老板觉得是他主动想给你涨 |
| **A** | 目标达成率 ≥ 70 | 成功拿到了合理涨薪或等值的有条件承诺 |
| **B** | 目标达成率 ≥ 50 | 涨薪没拿满，但方向对了，下次可继续推进 |
| **C** | 目标达成率 < 50 且 耐心 > 0 | 你被对方的节奏带着走，诉求基本落空 |
| **D** | 对方耐心归零（谈崩） | 谈话破裂，不管之前表现多好都是一票否决 |

> 💡 **最低底线**：不能带着"那我继续努力吧"结束这场谈话。即使没拿到数字，也要让老板给出一个有时限、可验证的承诺。

---

## ⚠️ 对手档案

| 属性 | 详情 |
|------|------|
| **姓名** | 张总 |
| **身份** | 部门总监，你的直属上级 |
| **性格标签** | 笑面虎 😊 — 表面温和关怀，句句埋刀 |
| **惯用话术** | "我也是为你好""公司今年不容易""你还年轻，先沉住气" |
| **核心弱点** | 怕数据、怕对比、怕员工真的离职 |

> 💡 **他不会直接骂你，不会拍桌子**。他的 PUA 是软刀子——让你自己觉得"好像提涨薪是我的问题"。

---

## 💡 战术提示

1. **前三轮别急着提数字**。先让老板说出"你今年表现不错"——让他自己铺垫，你再顺势推进。
2. **他说"我也是为你好"时，别急着反驳**。用"谢谢张总关心，我也想跟您汇报一下我今年的产出"接住话头，把话题拉回你的节奏。
3. **你的底牌是"今年三个爆款项目"**。每当你感觉被带跑时，用具体数据把话题拉回来。张总最怕这个。
4. **硬刚是一把双刃剑**。前三轮别用——等主动权积累到 50+ 再摊牌，效果翻倍。
5. **如果他说"公司今年效益不好"**，别接这个话茬。转聊"我今年给公司创造了什么价值"。

---

## 🚫 常见翻车姿势

| 翻车行为 | 后果 |
|----------|------|
| 第一轮就喊"我要涨薪 15%" | 张总直接切换防御模式，主动权暴跌 |
| 被"我也是为你好"带跑，开始聊人生规划 | 偏离涨薪主线，目标达成率持续下降 |
| 连续选学生思维"我理解公司不容易" | 张总顺势压价："你有这个觉悟很好" |
| 情绪上头连选三轮硬刚 | 笑面虎变冷暴力，耐心加速归零 |

## ⏱ 轮次提示

**共 10 轮对话**，建议节奏：

```
轮 1-3  → 别急着提钱，先摸清张总的态度
轮 4-7  → 核心战场，用你的策略推进涨薪
轮 8-10 → 摊牌收网，张总必须给你一个说法
```

每轮你有 4 张策略卡牌可选：🟢高情商 / 🟡老油条 / 🔵学生思维 / 🔴硬刚
