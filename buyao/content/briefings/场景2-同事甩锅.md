# ⚔️ 场景 2：同事甩锅到你头上

---

## 📍 场景描述

**时间**：周一上午 10 点，项目复盘会
**地点**：公司大会议室，三人——你、李明、王总
**你是谁**：项目组 UI 设计师，负责本次产品改版的设计交付
**发生了什么**：项目延期了两周，客户投诉到部门经理王总那里。复盘会上，你的同级同事李明（前端开发）突然抢先发言，把延期的责任全推到你头上——"设计稿一直改，我们开发根本没法做"。王总坐在中间，不表态，只是看着你。

---

## 🎯 你的目标

**让王总看清事实真相**，同时维护和李明的工作关系不至于彻底崩掉。

### 通关条件

| 等级 | 条件（数值阈值） | 含义 |
|------|-----------------|------|
| **S** | 目标达成率 ≥ 90 且 主动权 ≥ 70 | 王总完全认可你，李明收敛，你还展现了团队协作姿态 |
| **A** | 目标达成率 ≥ 70 | 王总认可你的说法或提出"设计环节不是延期主因" |
| **B** | 目标达成率 ≥ 50 | 锅没完全甩掉但也没全背上，关系没炸 |
| **C** | 目标达成率 < 50 且 耐心 > 0 | 你被李明压住了，王总虽然没有明确站队但倾向于相信李明 |
| **D** | 王总耐心归零（谈崩） | 和李明对喷到王总摔门而去——双输 |

> 💡 **最低底线**：不能默认背下这口锅。即使无法完全洗清，也要让王总看到你的证据和主动解决问题的态度——这会直接影响下次绩效评估。

---

## ⚠️ 对手档案

### 李明 —— 暴躁同事 🔥

| 属性 | 详情 |
|------|------|
| **身份** | 同部门平级，前端开发 |
| **性格标签** | 暴躁易怒 — 先发制人，声量压场 |
| **惯用话术** | "我早就说过""这不是我的问题""设计稿改来改去我们怎么做" |
| **核心弱点** | 怕证据（邮件/聊天记录/版本历史）、怕领导追问细节 |

### 王总 —— 观望领导 🧊

| 属性 | 详情 |
|------|------|
| **身份** | 部门经理，你和李明的共同上级 |
| **性格标签** | 沉着冷静 — 不站队，通过提问判断真相 |
| **惯用话术** | "你怎么看？""有证据吗？""这件事我大概了解了" |
| **核心决策逻辑** | 谁有证据信谁；如果都没证据，看谁更冷静、更在解决问题 |

> 💡 **不要跟李明对喷**。你的真正"用户"是王总——让他看到你是一个冷静、有证据、解决问题的专业人士。

---

## 💡 战术提示

1. **你的真正听众不是李明，是王总**。李明越激动，你越冷静——对比本身就是最好的反击。
2. **证据是最好的武器**。每一轮选卡牌时，优先选包含"我有邮件记录""聊天记录显示""版本历史证明"这类硬证据的选项。
3. **别被李明带节奏**。他说"就是你的问题"，别急着辩解"不是我的问题"——转而对王总说"王总，我来说明一下时间线"。
4. **适时展现团队意识**。即使李明在甩锅，你也要在适当时刻表示"我们后续可以优化流程"——这让王总看到你在解决问题而非制造对立。
5. **"老油条·引向流程"是针对王总的高效牌**——把焦点从"谁的责任"转移到"怎么避免下次"，王总好感度显著提升。

---

## 🚫 常见翻车姿势

| 翻车行为 | 后果 |
|----------|------|
| 跟李明对喷"你才在推卸责任" | 两人一起在领导面前失分，耐心暴跌 |
| 被李明气势压倒，选学生思维 | 李明乘胜追击，主动权 -15 |
| 情绪激动但拿不出证据 | 王总判定双方各打五十大板 |
| 全程只对李明说话，完全忽略王总 | 错过了影响决策者的黄金机会 |
| 得理不饶人，把李明逼到墙角 | 王总虽然认可事实但觉得你不会做人 |

---

## 👥 多人规则

每轮发言顺序：
1. **李明**先开火（甩锅/辩解/情绪输出）
2. **王总**简短回应或提问
3. **你**出牌回应 —— 卡牌会标注「对李明说」「对王总说」或「对两人说」

---

## ⏱ 轮次提示

**共 10 轮对话**，建议节奏：

```
轮 1-3  → 李明的火力最猛，稳住心态，别急着反击
轮 4-7  → 找到李明的逻辑漏洞，用你的策略引导王总提问
轮 8-10 → 王总的判断基本成型，李明可能认怂或变本加厉
```
