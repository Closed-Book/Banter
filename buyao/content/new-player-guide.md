# 新手引导文案

> Demo 首次加载时展示，帮助评委/用户 30 秒内理解产品怎么玩。
> P1 渲染第一屏时覆盖半透明引导层，点击"知道了"关闭。

---

## 第一屏引导（场景选择页）

```
🎮 欢迎来到「别把天聊死」

这里是一个安全的社交训练场——你可以在里面"练"那些现实里让你头疼的高压对话。

📋 怎么玩？很简单：
① 选一个你想练习的场景（比如跟老板谈涨薪）
② 回答 3 个简单问题，系统会定制你的角色
③ 进入对话——你不需要自己打字，从 4 张策略卡牌里选一张出牌
④ NPC 会像真人一样回应你，有情绪、有套路、有弱点
⑤ 10 轮对话后，系统给你打分——从 S 到 D，附带改进建议

🎯 目标不是"赢"，是让你在现实中再遇到这种对话时，知道自己可以怎么说。

💡 提示：MVP 版本已开放「老板PUA压薪」和「同事甩锅」两个场景。其余标注"即将开放"。
```

---

## 第三屏引导（Briefing 后、第一轮对话前）

```
⚔️ 进入副本！

对方已经就位。每轮你需要从 4 张卡牌中选一张出牌：

🟢 高情商 —— 认可对方但坚持自己的立场
🟡 老油条 —— 打太极、转移焦点、拖时间
🔵 学生思维 —— 老实听话、容易被拿捏（⚠️ 通常会让步）
🔴 硬刚 —— 直接摊牌，不留面子（⚠️ 谨慎使用）

📊 顶部三个数值条会实时变化：
- 主动权：谁在掌控对话节奏（越高你越主动）
- 对方耐心：降到 0 = 谈崩 = D 级结局
- 目标达成率：你的诉求实现了多少

点击卡牌查看完整话术 → 确认出牌 → NPC 回应 → 进入下一轮。
```

## 结局页引导

```
📊 你的战报

- 等级 S/A/B/C/D —— 一看就懂
- 五维雷达图 —— 表达力/共情力/策略性/抗压力/边界感
- 最佳 & 最危险的一句话回放 —— 记住什么该说、什么不该说
- 现实行动建议 —— 下次真的面对这种对话，你该怎么开口

💡 建议：同一场景玩 2-3 次，试试不同的出牌策略，对比结局差异。
```
