# 结局评分逻辑

> 对齐 `shared/contract.ts`，P2 写 ending prompt 时照此执行。

---

## 一、等级判定（优先级从高到低）

| 优先级 | 等级 | 条件 |
|--------|------|------|
| 1 | **D** | `patience = 0`（谈崩，一票否决——无论其他数值多高） |
| 2 | **S** | `goal_progress ≥ 90` 且 `initiative ≥ 70` |
| 3 | **A** | `goal_progress ≥ 70` |
| 4 | **B** | `goal_progress ≥ 50` |
| 5 | **C** | 以上都不满足（兜底） |

> ⚠️ D 级一票否决意味着：即使 goal_progress = 95、initiative = 90，只要 patience 归零就是 D。
> 必须先检查 patience，再检查 S/A/B/C。

---

## 二、五维雷达怎么算

P2 在 ending prompt 里让模型根据**用户 10 轮的出牌记录**直接打分。给模型的指令：

### 通用规则（所有场景适用）

```
根据以下规则给五维打分（0-100，整数）：

1. 表达力(expression)：用户是否把诉求说清楚了
   - 基础分 = goal_progress 最终值
   - 选「高情商」≥3次 → +10
   - 选「学生思维」≥4次 → -15分

2. 共情力(empathy)：用户是否理解对方但不被带着走
   - 基础分 = 60
   - 选「高情商」≥3次 → +10
   - 选「硬刚」≥4次 → -15
   - patience 最终值 ≥50 → +5

3. 策略性(strategy)：用户是否灵活切换打法
   - 基础分 = 50
   - 四种卡牌都用过 → +20
   - 只用过1-2种 → -15
   - 主动权 ≥70 → +10

4. 抗压力(resilience)：被施压后是否稳住了
   - 基础分 = initiative 最终值
   - 被施压后下一轮主动权反弹 → 每次 +5
   - 连续2轮以上主动权下降 → 每段 -10

5. 边界感(boundary)：是否既争取了利益又没撕破脸
   - 基础分 = 50
   - 选「硬刚」≥5次 → -20
   - 选「学生思维」≥4次 → -20
   - goal_progress ≥70 且 patience ≥40 → +20
```

### 场景2 专属调整

在通用规则基础上，场景2 增加以下修正：

```
场景2 附加规则：

3. 策略性(strategy)：
   - 选「对王总说」的卡牌 ≥3 次 → +10（说明用户认准了决策者）
   - 全程只选「对李明说」的卡牌 → -10

5. 边界感(boundary)：
   - 李明主动收敛/认错 → +10
   - 跟李明对喷超过 2 轮 → -15
   - 王总在总结中表扬了用户"团队意识" → +10
```

> P2 也可以选择直接让模型根据对话历史主观打分，以上规则作为校验兜底。

## 三、best_line / worst_line 选取

| 字段 | 规则 |
|------|------|
| **best_line** | 选用户出牌后 `goal_progress` 增加最多的那一轮 |
| **worst_line** | 选用户出牌后 `initiative` 下降最多的那一轮 |
| 附带 | `round` 编号 |

### 平局处理

- best_line 平局 → 取**较晚**的轮次（更能代表最终能力）
- worst_line 平局 → 取**较早**的轮次（早期翻车对结局影响更大，也更值得复盘）
- 如果全程 delta 均为 0（极端情况） → best_line 取最后一轮，worst_line 取第一轮

| 等级 | one_liner 示例 | real_advice 示例 |
|------|---------------|-----------------|
| S | "不拍桌子也能把事谈成，这才是高手。" | "记住今天的感觉——冷静不是退让，是控场。" |
| A | "没拿满，但守住了底线。" | "下次带3个可量化业绩数据进去，第一句话就拿主动权。" |
| B | "有惊无险过了，但本来可以更好。" | "中间几轮容易掉节奏，盯紧你的核心诉求别被带跑。" |
| C | "你被对方吃得死死的——这就是被PUA的感觉。" | "听到'我也是为你好'先停三秒再回应，别急着接受对方的框架。" |
| D | "谈崩了。沙盘里崩总比现实里崩强。" | "硬刚是最后一张牌。前三轮先用高情商摸清底线，再决定要不要摊牌。" |

| 等级 | one_liner 示例 | real_advice 示例 |

| A | "锅没背上，关系也没炸，稳稳过关。" | "下次在会议前就把关键证据准备好，开场第一句话定调。" |

## 五、结局称号

| 等级 | 称号规则 | 场景1 示例 | 场景2 示例 |
|------|---------|-----------|-----------|
| S | 「[褒义形容词]的[角色]」 | "不卑不亢的谈判者""冷静有力的职场人" | "四两拨千斤的控局者""证据在手的清白人" |
| A | 「稳扎稳打的[角色]」 | "稳扎稳打的沟通者" | "守住底线的合作者" |
| B | 「[中性词]的[角色]」 | "有惊无险的过关者" | "勉强脱身的接锅侠" |
| C | 「被[动词]的[角色]」 | "被压制的妥协者" | "被带偏的背锅人" |
| D | 「[动词]翻车现场」 | "硬刚翻车现场""谈判破裂者" | "对喷翻车现场""双输制造者" |

### 多人场景称号特殊规则

- 如果 D 级且李明的出格行为触发了王总对李明的不满 → "虽败犹荣——李明比你更惨"
- 如果 B 级且王总最后的总结偏向用户 → 可以给 A 级称号，表彰证据意识
