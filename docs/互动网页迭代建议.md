# 互动网页迭代建议

## 第一张：办公室咖啡厅面试

当前目标：打开网页后直接进入陈总和“我”面对面的可交互对话，用户可以阅读陈总台词、查看隐藏历史、选择四张回应卡牌并推进回合。

## 暂存建议

1. 重新生成一版更明确的“从我的左肩看过去”的前景图，让肩线和视角更贴近用户描述。
2. 为“我”补充不同情绪姿态，例如紧张、认真、放松、迟疑，用于后续回合切换。
3. 为陈总生成第2到第10张表情图，替换现在的单张背景或做局部表情过渡。
4. 卡牌 hover 可以继续加强游戏感，例如增加边缘粒子、轻微旋转、音效接口或稀有度色阶。
5. 移动端可以单独做竖屏构图，减少前景人物和卡牌之间的拥挤。
6. 左侧历史抽屉后续可以增加“关键节点”标记，帮助用户回看自己哪句话影响了局势。
7. 对话框可以加入逐字显示、等待停顿和镜头轻微推拉，让面试压迫感更明显。
